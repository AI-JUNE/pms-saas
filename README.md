# PMS — Project Management System (SaaS) v4 · Powered by GOWON

멀티테넌트 상용형 PMS. Next.js + Drizzle + Postgres. eum 테라코타 프리미엄 UI.

## 기능
대시보드(차트) · 프로젝트/단계 · 업무(WBS, 칸반+간트 타임라인) · 요구사항 · 산출물/결재 ·
이슈/결함(칸반·라벨) · 리스크(히트맵) · 회의 · 인력 · 리포트(번다운 등) ·
엔티티 댓글(@멘션 알림) · CSV 내보내기 · ⌘K 커맨드 팔레트 ·
관리자(사용자·권한) · 감사 로그 · 조직 설정 · 검색/필터/정렬 · 상세 슬라이드 패널.

## 시작
npm install → .env에 DATABASE_URL → npm run db:push → npm run db:seed → npm run dev
데모: admin@demo.local / admin1234
